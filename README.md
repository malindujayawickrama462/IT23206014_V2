# IT23206014_V2

Playwright tests for SwiftTranslator (https://www.swifttranslator.com/) and the Playwright docs example.

## Setup

1. Install dependencies: `npm install`
2. Install Playwright browsers (required before first run): `npx playwright install`

## Run tests

- `npm test` — run all tests
- `npm run test:headed` — run with browser visible
- `npm run test:ui` — run with Playwright UI
